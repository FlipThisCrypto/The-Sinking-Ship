Non-negotiables

Every PR and art batch should respect these

1. Fairness first

Identical inputs → byte-identical manifests. Commitment before mint; salt reveal after. Never pre-reveal traits before payment settles.

2. Config drives code

No hardcoded trait names, prices, or odds in Python. Change config/, re-validate, re-simulate.

3. Art grammar

Bone-white ground, vertical warm→navy ink, dense calligraphic linework, crystals/filigree. Match ships_amano/ and Tom Bepe refs — never Pepe IP.

4. Testnet first

Anything that touches chain: testnet11 dry run before mainnet. Dry-run flags on fulfillment.

5. Spec > improvisation

Ambiguity → open question / ADR. Owner rulings recorded (OQ-1, OQ-2, OQ-4, ADR-0008, STM surface).

6. Thesis

Builders create value after everyone leaves. Marketing is conviction and salvage, not hopium or guaranteed returns.

How to work this project

Practical instructions for engineers & art agents

A. Engine day

  1. pip install -r requirements.txt
  2. python engine/validate_configs.py
  3. python engine/simulate.py --profile sellout --seed check --replicates 5 --check
  4. pytest — determinism is sacred
  5. Fulfillment dry run: see docs/TODO-P7-fulfillment.md CLI

B. Art day

  1. Read docs/art-reference/ART-DIRECTION.md + CLAUDE-STYLE-PACK.md
  2. Use ships_amano/ and Tom Bepe refs as style floor (don’t redistribute as mint art)
  3. Replace sprites/<layer>/*.png file-for-file at illustration master size
  4. python scripts/style_loop.py or score samples vs golden set
  5. Landing art lives in site/assets/ (depths, passes, chest)

C. Landing / demo day

  1. Local: python -m http.server 8471 --directory site
  2. Push site/** to main → GitHub Actions deploys Pages
  3. Public: landing
  4. This page: dashboard.html on the same host

D. Suggested build order

  1. Done path: P2 → P3 → P4 → P5 → P6
  2. Now: finish art layers + P7 live chain path
  3. Then: P1 lore → P10 copy → expand P9 fairness docs
  4. Then: P8 reveal → P11 badges → P12 launch checklist
  5. Never mainnet until testnet11 go/no-go is green

E. Owner decisions still open

  • OQ-8: final CHIP-0007 collection id format
  • OQ-9: series_total vs Scuttling realized supply messaging
  • Project DID + royalty address in config/collection.json
  • Final layer art production plan / budget for ~300 illustration assets
  • Wizard of the Deep puzzle design (marketing centerpiece)

F. Showing people

  1. Landing for mood, tiers, art direction
  2. This dashboard for “where we are / what’s next”
  3. Repo README + ADRs for technical buyers
  4. Sample renders under output/ (gitignored — generate locally)
