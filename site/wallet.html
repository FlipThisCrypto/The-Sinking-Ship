Get ready to dive

Blind mint uses Chia offer files. You pay with a Dive Pass offer; you claim your chest with a 0-XCH claim offer. No custodial deposit address.

  1. Install Sage — desktop wallet with WalletConnect. Approve chia_takeOffer and chia_signMessageByAddress as required namespaces (Sage ignores optionals).
  2. Fund testnet11 first — every mint dry-run happens on testnet before mainnet. Never skip this.
  3. Take the Dive Pass offer — pays fixed XCH for your tier. Wait for on-chain confirmation (not just the wallet toast).
  4. Claim the chest — after fulfillment, accept a 0-XCH claim offer for the rolled NFTs. If take result is “unclear,” treat as pending — never double-click panic.
  5. Fallback — download the offer file and import into Sage if WalletConnect fails.
  6. Verify — after salt reveal, run chest_roller.py verify or the fairness page self-check.
