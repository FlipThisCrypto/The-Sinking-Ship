# Art reference

Canonical style references for THE SINKING SHIP. Files here are **inputs to
art direction**, not generation assets — nothing in this folder is consumed by
the render engine. Read [ART-DIRECTION.md](ART-DIRECTION.md) for the full
style brief and the open medium decision (OQ-11).

## Contents

- **[ART-DIRECTION.md](ART-DIRECTION.md)** — the authoritative look: Amano-style
  aquatic ink illustration, gradient-encodes-depth colour system, Tom Bepe
  character sheet, trait-layer mapping, and the pixel-vs-illustration fork.
- **`pirate-ship/`** — 12 owner-supplied ship references (gradient ink line-art
  + full watercolour). The model for ships, seas, skies, and the ornate
  filigree/wave motif. These replace the flat placeholder backgrounds.
- **`tom-bepe-amano/`** — 14 Tom Bepe references (9 smoking + 5 "no smoke").
  The character sheet source: frog-man everyman, black Amano hair, gold
  filigree collar, big expressive eye, dissolving-into-water lower body.

## Rules (spec Section 9, Risk 2)

Bepe-inspired but **legally distinct** — original design, no traced
silhouettes, no protected marks, distinct proportions/features. Adapt mood,
motif, and palette; never reproduce a specific reference. If a reference
carries third-party rights or a watermark, it is not shippable as project art.

`config/palette.json` (v2.0.0) is derived from these references.
